# Flappy (Development Branch)

Flappy is the package manager for **FlucidOS**.

This branch (`dev`) is the **active development branch**.
It may contain incomplete trails, breaking changes, or internal refactors.

Do NOT treat this branch as stable.

---

## Current Trail

- Trail 2 is complete and locked
- Future trails will be developed incrementally on this branch

Each trail is designed to:
- be minimal
- be verifiable
- lock cleanly before moving on
